I see it: your lab print containers are sized to 297 mm height plus the browser’s print margins/padding, so they overflow by a few millimetres and Chrome pushes a second blank page. You’re also using min-h-[100vh], which prints taller than A4. 
 Your PDF shows exactly that 2-page spill. 

Do these two fixes and it’ll be one page, every time.

1) Replace the lab print container classes

In Laboratory.tsx, change both print wrappers to a single class that we’ll define:

Before (request & report):

<div
  id="lab-request-print"
  className="flex flex-col min-h-[100vh] print:min-h-[100vh] print:w-[210mm] print:h-[297mm] p-8"
>
...
<div
  id="lab-report-print"
  className="flex flex-col min-h-[100vh] print:min-h-[100vh] print:w-[210mm] print:h-[297mm] p-8"
>


After:

<div id="lab-request-print" className="rx-print">
...
<div id="lab-report-print" className="rx-print">


(Leave the inner content as is.) 

2) Add these print rules to your global CSS (e.g., index.css)
/* Shared one-page A4 container */
.rx-print{
  width:210mm;
  min-height:calc(297mm - 24mm);  /* 12mm top + 12mm bottom margins */
  padding:10mm;                   /* inside the A4 box */
  box-sizing:border-box;
  display:flex;
  flex-direction:column;
}
.rx-print .mt-auto{ margin-top:auto !important; } /* pin footer to bottom */

@media print{
  @page{ size:A4; margin:12mm; }
  html,body{ margin:0; padding:0; }
  body *{ visibility:hidden; }

  /* Show ONLY whichever print node is mounted */
  #lab-request-print, #lab-request-print *{ visibility:visible; }
  #lab-report-print, #lab-report-print *{ visibility:visible; }

  #lab-request-print,
  #lab-report-print{
    position:absolute; inset:0;
    overflow:hidden;                /* avoid spill causing page 2 */
    page-break-before:avoid;
    page-break-after:avoid;
  }

  .no-print{ display:none !important; }
}


Why this works:

We remove min-h:100vh and the fixed print:h:297mm that caused overflow.

We size the card in millimetres inside the @page margins, so total height stays < 297 mm.

We hide everything except the single active print container.
